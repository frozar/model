Chapitre 7. Petit léxique

Par ordre alphabétique :

  • BAL = Boite aux lettres;

  • DNS = Domain Name Service (base de données IP - nom);

  • FAI = Fournisseur d'Accès à l'Internet;

  • MDA = Mail Delivery Agent (par exemple deliver, qmail-local, procmail);

  • MTA = Mail Transport Agent (par exemple sendmail );

  • MUA = Mail User Agent (par exemple elm, pine ou mutt);

  • MX = Mail Exchanger. Type d'enregistrement DNS;

  • NFS = Network File System;

  • POP3 = Post Office Protocol version 3 (port TCP 110);

  • QMTP = Quick Mail Transport Protocol (port TCP 209);

  • RCF 822 = Request For Comment n°822 (standardisant le courriel);

  • SMTP = Simple Mail Transport Protocol (port TCP 25);

  • TCP = Transmission Control Protocol;

  • UCSPI = Unix Client-Server Program Interface

  • UUCP = Unix to Unix CoPy

  • maildir = Format de boite aux lettres reposant sur des répertoires. Un fichier pour chaque courriel. Faire «man maildir»;

  • mbox = Format de boite aux lettres reposant sur un seul fichier. Les courriels sont regroupés dans un seul et unique fichier. Faire «man mbox».
