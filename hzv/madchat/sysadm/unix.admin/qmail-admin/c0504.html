Fetchmail

Pour récupérer votre courriel sur le port POP3 de votre FAI, je vous conseille d'utiliser l'exellent fetchmail. Voici un exemple de ~/.fetchmailrc possible : 
poll mail.monfai.amoi protocol pop3:
 username monpseudo
 password monsesame
 fetchall
 batchlimit 1 # utile si le FAI utilise qmail
 is loginlocal here
 mda "env -u QMAILINJECT qmail-inject -f%F %T"
fetchmail fera donc appel à qmail-inject pour insérer les courriels dans la file d'attente de votre qmail. qmail se chargera ensuite de livrer le courriel dans votre BAL via le MDA qmail-local.

Vous pouvez aussi décider d'utiliser plutôt votre propre port SMTP pour l'insertion dans la file d'attente de votre qmail. Voici le ~/.fetchmailrc correspondant : 
poll mail.monfai.amoi protocol pop3:
 username monpseudo
 password monsesame
 fetchall
 batchlimit 1 # utile si le FAI utilise qmail 
 forcecr # RFC 822 strict
 is loginlocal here

En fait, l'insertion dans la file d'attente de votre qmail ne sert pas à grand chose. Autant faire appel directement à un MDA qui va directement livrer le courriel dans votre BAL. Le plus simple est par exemple d'utiliser procmail. On courcircuite qmail car il ne sert à rien ici ! Voici le ~/.fetchmailrc correspondant : 
poll mail.monfai.amoi protocol pop3:
 username monpseudo
 password monsesame
 fetchall
 batchlimit 1 # utile si le FAI utilise qmail
 is loginlocal here 
 mda "procmail -d loginlocal"
