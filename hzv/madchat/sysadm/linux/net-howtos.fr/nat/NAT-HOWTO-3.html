3. Les Deux Types de NAT

Je divise le NAT en deux types différents : le NAT Source (SNAT) et le NAT Destination (DNAT).

Le NAT Source c'est quand on altère l'adresse source du premier paquet : par exemple. Tu changes l'endroit dont la connection provient. Le NAT Source est toujours réalisé après le routage, juste avant que le paquet ne sorte sur la ligne. Le Masquerading est une forme spécialisée de SNAT.

Le NAT Destination c'est quand on altère l'adresse de destination du premier paquet : par exemple. Tu changes l'endroit où la connection va aboutir. Le NAT Destination est toujours effectué avant le routage, quand un paquet arrive en premier sur la ligne. Le Port forwarding, load-sharing et le proxy transparent sont tous des formes de DNAT.
