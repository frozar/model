Ca consiste à passer par un proxy (squid pour ne pas le nommer)
configuré pour passer toutes les pages à un script qui va se charger
de les expurger.

Donc il faut deux trucs:
* Squid (www.squid-cache.org). Ca se compile les doigts dans le nez sur
toutes les distribs.
* Adzap (www.zip.com.au/~cs/adzap/). C'est un script, rien à compiler
donc (faut juste rajouter une directive dans le squid.conf, de toute
manière tout est expliqué sur la page).

Voilà. Avec ca, vous pourrez enfin visiter les pages multimania sans
vous taper tous les popups de merde (parce que le script ferme les
popups tout seul).
En plus le script est facile à updater pour virer de nouvelles pubs.

Le seul inconvénient que j'ai constaté jusque là: il a l'air de mal
digérer le AD de adobe.com, et il zappe la moitié des graphismes.
C'est pas grave, perso je glande pas des heures sur adobe.com. :)
