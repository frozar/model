Référence des commandes AT fax

Le tableau suivant dresse une liste des commandes AT fax que votre modem comprend. Chaque description nomme la commande, l'explique et donne la liste des paramètres associés.

IMPORTANT : Votre modem doit fonctionner sous Microsoft Windows 95. Il ne fonctionne pas sous MS-DOS ou les versions antérieures de Microsoft Windows.

Commandes fax
Commande Description
+F<commande>=? Indication des possibilités de fonctionnement. AT+F<commande>=? détermine les possibilités de fonctionnement du modem.

Réponses :

+FCLASS = ? 0, 1, 8, 80

+FTM = ? 3, 24, 48, 72, 96, 73, 74, 97, 98, 121, 122, 145, 146

+FRM = ? 3, 24, 48, 72, 96, 73, 74, 97, 98, 121, 122, 145, 146

+FTH = ? 3, 24, 48, 72, 96, 73, 74, 97, 98, 121, 122, 145, 146

+FRH = ? 3, 24, 48, 72, 96, 73, 74, 97, 98, 121, 122, 145, 146
+F<commande>=n? Indication des paramètres pris en charge. AT+F<commande>=n? demande au modem si cette fonction est supportée. Le mode renvoie la réponse OK si le paramètre de cette commande est pris en charge, et la réponse ERROR dans le cas contraire.
+FCLASS? Indiquer la configuration active. AT+FCLASS? demande au modem de déterminer la configuration active.

Réponses :

0 = Mode données

1 = Mode fax

8 = Mode voix

80 = Mode VoiceView
+FCLASS=n Sélectionner la catégorie de service. AT+FCLASS= sélectionne la catégorie de service souhaitée.

Paramètres : n = 0, 1, 8, 80

n = 0 Mode données (par défaut)

n = 1 Fax Classe 1

n = 8 Mode voix

n = 80 Mode VoiceView
+FTS=n Interrompre la transmission et attendre. AT+FTS oblige le modem à arrêter la transmission de données et à attendre n intervalles de 10 millisecondes avant de répondre par le code de résultat OK.

Paramètres : n = 0 - 255 (intervalles de 10 millisecondes) (valeur par défaut = 0)

Un code de réponse ERROR est émis si cette commande est entrée alors que le modem est déconnecté.
+FRS=n Recevoir le temps de silence. AT+FRS force le modem à attendre pendant n intervalles de silence de 10 millisecondes sur la ligne, puis retourne un code résultat OK à l'ordinateur.

Paramètres : n = 0 - 255 (intervalles de 10 millisecondes) (valeur par défaut = 0)

Cette commande est abandonnée si un caractère est reçu. Le modem supprime le caractère d'abandon et émet le code de résultat OK.

Un code de réponse ERROR est émis si cette commande est entrée alors que le modem est déconnecté.

n = 48 V.27ter, 4800 bps (par défaut)

n = 72 V.29, 7200 bps

n = 73 V.17, 7200 bps

n = 74 V.17, 7200 bps (synchronisation abrégée)

n = 96 V.29, 9600 bps

n = 97 V.17, 9600 bps

n = 98 V.17, 9600 bps (synchronisation abrégée)

n = 121 V.17, 12000 bps

n = 122 V.17, 12000 bps (synchronisation abrégée)

n = 145 V.17, 14400 bps

n = 146 V.17, 14400 bps (synchronisation abrégée)

Un code de réponse ERROR est émis si cette commande est entrée alors que le modem est déconnecté.
