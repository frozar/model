GLSL Hacker Live Updaters
-------------------------

network_live_updater_glsl.exe and network_live_updater_lua_python.exe are 
Windows 64-bit applications. You have to use Wine (http://www.winehq.org)
to execute both applications:

$ wine network_live_updater_glsl.exe

$ wine network_live_updater_lua_python.exe
